# ENUNPUNTO

ENUNPUNTO website implementation for review.

## Stack

- Next.js App Router
- TypeScript
- React
- Custom CSS
- Bilingual content: English and Spanish

## Local Commands

```bash
npm install
npm run lint
npm run build
npm run dev
```

## Notes

- Default English route: `/`
- Spanish route: `/es`
- Contact delivery is prepared for a future provider through environment variables.
- No deployment has been configured or executed from this project.
